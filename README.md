A content-based music recommendation system built using Spotify audio features and cosine similarity.

## Overview
This project recommends songs based on audio similarity using a content-based filtering approach.
Custom evaluation metrics were used to assess recommendation quality beyond traditional accuracy.

## Techniques Used
- Content-Based Filtering
- Cosine Similarity
- Scikit-learn Pipelines
- Feature Scaling & Imputation

## Evaluation Metrics
- Precision@K
- F1-score
- Per-Genre Overlap
- Random Baseline Comparison

## Key Results
- Achieved significant improvement over random recommendations
- Strong genre-wise consistency for Pop and EDM tracks

## Limitations
- No collaborative filtering
- No personalization using user history

## Future Work
